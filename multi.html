This project utilizes a PIC24F16KA101 microcontroller in order to make a multimeter with both voltage and resistance measurement capabilities. The microcontroller is programmed in C using the MPLAB X IDE and the MPLAB XC16 compiler with a PICkit 4. The values are read from potentionmeters and displayed onto the terminal via a UART connection. The values can then read by a Python script which records 10 seconds of data and displays them onto graphs.
